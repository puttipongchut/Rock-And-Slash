# Action RPG Side-scrolling Game

This is a classic 2D action adventure game inspired by the likes of Hollow Knight and Dark Souls, this game is developed in Java Swing Framework for EGCO213 Programming Paradigms Course's final project.

## Features
- In the game, the player can attack up to 3 combos, roll to dodge enemy attacks, and self-heal (limited).
- Face many powerful foes. Can you defeat them all?
- An in-game timer for capturing your best time to finish the game.
- An unique and powerful enemy in the last level, can you beat the Golem?

## Get Started
- Make sure you have JRE/JDK Version 21 or later installed.

## How to run the game
### First clone this repository
```
git clone https://github.com/puttipongchut/Rock-And-Slash.git
```
### Next, You can either
- Run the game in an IDE like Netbeans or Intellij (Java 21 or older) via MainApplication.java
- Or run the game directly from the .jar file in out/artifacts/game_jar/game.jar
